# スライド資料：汎用コンサルティングスライド生成システムの構築

## このプルリクエストは何をしているか？

**2つのことを行っています：**

1. GenAI動画生成モデル市場の分析スライド（12枚）をHTML形式で作成しました
2. **どんなテーマでも使える汎用スライド生成システム**を構築しました — ユーザーがテーマとデータを渡すだけで、McKinsey/BCGスタイルのスライドが自動生成されます

---

## なぜこの変更が必要か？

毎回ゼロからスライドのデザインを指示するのは非効率です。
そこで、**1回の設定で何度でも使える仕組み**を作りました：

```
【従来のフロー】
ユーザーが毎回デザイン指示を書く → LLMが生成 → バラバラな品質

【今回構築したフロー】
システムプロンプトを1回設定 → ユーザーはテーマとデータだけ送る → 統一品質のスライドが毎回生成される
```

## システムプロンプトの主な改善点（v1 → v2）

| 項目 | v1（初回） | v2（今回のアップデート） |
|------|-----------|----------------------|
| 対応テーマ | GenAI動画市場のみ | **あらゆるテーマに対応** |
| テンプレート | 1種類（市場分析） | **4種類**（市場分析、技術トレンド、事業提案、簡易レポート） |
| 言語 | 英語のみ | **多言語対応**（入力言語に自動追従） |
| データ→チャート変換 | 手動で指定 | **自動選択ルール**（11種類のデータパターンに対応） |
| 品質管理 | なし | **10項目のチェックリスト**を追加 |
| 使用例 | なし | **4つの使用例**（AI市場、SaaS、社内提案、URL貼り付け） |
| チャート種類 | 5種類 | **11種類**（積み上げ棒、円グラフ、タイムライン等を追加） |
| 説明言語 | 英語 | **日本語**（説明・コメントすべて日本語化） |
| FAQ | なし | **よくある質問4項目**を追加 |

## 汎用プロンプトの使い方（簡単3ステップ）

```
ステップ1: prompts/system-prompt.md の中のシステムプロンプト部分をLLMに設定する
ステップ2: テーマとデータを送る（例：「日本のSaaS市場を分析して」）
ステップ3: 出力されたHTMLを保存してブラウザで開く → 完成！
```

### 入力例

```
テーマ: 日本のSaaS市場 2026年版
データ: [記事やレポートの内容をそのまま貼り付け]
要件: 8枚程度、日本語、テンプレートAを使用
```

## 動作確認の方法

1. `slides/genai-video-market-deck.html` をChromeまたはEdgeで開く
2. 12枚のスライドが縦にスクロールで表示されることを確認
3. `Ctrl + P`（Macは `Cmd + P`）で印刷プレビューを開き、PDFとして保存できることを確認
4. フォント（Playfair Display、Inter）が正しく読み込まれていることを確認
5. 各スライドが16:9（1280×720px）の比率で表示されていることを確認
6. `prompts/system-prompt.md` をLLMに渡して、別のテーマでスライドが生成できることを確認

## 用語の補足（新卒の方向け）

- **システムプロンプト**: LLM（AIチャットツール）に対して「あなたの役割はこれです」と最初に設定する指示文のことです。ユーザーの質問とは別に、AIの振る舞いを決めるために使います。
- **McKinsey/BCGスタイル**: 世界的な経営コンサルティング会社の資料スタイル。データ重視で、各スライドに「アクションタイトル」（結論を述べる見出し）を付けるのが特徴です。
- **ピラミッド原則**: コンサルティング資料の構成法。結論を最初に述べ、その後に根拠を並べる「結論ファースト」の考え方です。
- **アクションタイトル**: 「市場概要」のような単なるラベルではなく、「SaaS市場は年率25%で成長し2028年に2兆円に到達」のように、そのスライドの結論を1文で述べる見出しのことです。
- **ハーベイボール**: コンサルティング資料でよく使われる、丸の塗りつぶし度合いで評価を示す記号です（●◕◑◔○）。
- **2×2マトリクス**: 2つの軸で4象限に分けて、各プレイヤーの位置づけを可視化するフレームワークです。
- **テンプレートA/B/C/D**: システムプロンプト内で定義されている4種類のスライド構成パターンです。テーマに応じて最適なものが自動選択されます。
